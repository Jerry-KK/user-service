# 签到系统
### 背景：
- 100万用户，签到送积分
- 连续签到7天赠送10积分

### 演进:
- V1: MySQL 单表阶段(用于跑通业务逻辑)
- V2: MySQL 按月分表 (解决单表大数据量过大)
- V3: Redis BitMap 方案 (空间与性能优化)
